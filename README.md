# Dexterity Capstone 1 — DexConnect

![Flutter](https://img.shields.io/badge/Flutter-02569B?style=flat&logo=flutter&logoColor=white)
![Dart](https://img.shields.io/badge/Dart-0175C2?style=flat&logo=dart&logoColor=white)

**DexConnect** — A Flutter application that displays opportunities and programs from the Dexterity platform, including career development programmes, essay contests, and article writing competitions. Built as a capstone project during the **Dexterity internship program**.

## Features

- **Opportunity Cards** — Browse programs with descriptions and details
- **Delete Functionality** — Remove cards from the list
- **Material Design** — Clean, modern UI with quote-card style layout

## Getting Started

```bash
git clone https://github.com/Sanjays2402/Dexterity-capstone-1.git
cd Dexterity-capstone-1
flutter pub get
flutter run
```

## Project Structure

```
├── lib/
│ ├── main.dart # App entry & QuoteList widget
│ ├── quote.dart # Quote data model
│ └── quote_card.dart # Quote card UI widget
├── ios/ # iOS platform files
└── pubspec.yaml
```
